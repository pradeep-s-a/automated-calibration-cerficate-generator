# automated-calibration-cerficate-generator

This project tries to develop a calibration system which automates the calibration process and provides digital certification for authenticity.

The project main advantage is it's cross platform compatability which is achieved by using only dependencies which are platfform independent. The system is implemented using Fluke 5502a multiproduct calibrator which generates the reference signals and YTA610 temperature transmitter which acts as IUT (instrument under test).

The system is lightweight and can be run even in a lower model Raspberry Pi.
